# Room 1221 - User Guide & Feature Overview

## 🎯 Quick Start

### First Time Users
1. **Open the app** - You'll see a welcome modal
2. **Set a nickname** (optional) - Or click "Chat Anonymously"
3. **Start chatting** - Ask any SRHR question
4. **Explore features** - Use the menu (☰) to access all sections

### Returning Users
- Your nickname and preferences are remembered
- Chat history is available (within your retention period)
- Language and settings are preserved

## 🔒 Privacy Features

### Anonymous Chat
- **No login required** - Start chatting instantly
- **Optional nickname** - Personalize without revealing identity
- **No personal data** - We never collect identifying information
- **Local storage only** - Everything stays on your device

### Auto-Delete Settings
Choose how long to keep your chat history:
- **24 hours** (default) - Good for maximum privacy
- **7 days** - Remember conversations for a week
- **30 days** - Month-long retention
- **90 days** - Extended access

**How it works:**
- Timer starts when you begin a chat
- Progress bar shows time remaining
- Chats automatically delete after the period
- You can manually clear anytime

### Session Timer
Located in the sidebar, shows:
- Time remaining until auto-delete
- Visual progress bar
- Current retention period

### Panic Exit
**Emergency privacy feature:**
- Tap the red "Quick Exit" button in header
- App instantly switches to a safe screen (calculator)
- Returns to chat when you tap "Return to Room 1221"
- Use when you need quick privacy

## 💬 Chat Features

### Text Input
- Type your question in the input field
- Press Enter or tap Send button
- Questions can be in any supported language

### Voice Input 🎤
**How to use:**
1. Tap the **microphone icon** in the input field
2. Allow microphone access (first time only)
3. Speak your question
4. Text appears automatically
5. Tap Send

**Supported voice languages:**
- English (en-US)
- Twi (tw-GH)
- Ewe (ee-GH)

**Troubleshooting:**
- Ensure microphone permissions are granted
- Works best in Chrome/Edge browsers
- Requires internet connection
- Icon turns red when listening

### Quick Replies
First-time users see suggested questions:
- "Tell me about contraception"
- "What are STIs?"
- "Period questions"

Tap any suggestion to auto-fill the input.

### Typing Indicator
When Room 1221 is composing a response:
- Three animated dots appear
- Located in a chat bubble
- Shows "Room 1221 is thinking..."

### Chat Actions
**Clear Chat:**
1. Open sidebar menu
2. Tap "Clear Chat" under Actions
3. Confirm in the dialog
4. All messages deleted instantly

## ⚙️ Settings & Privacy

### Your Nickname
- **Current nickname** displayed
- **Change anytime** via Settings
- **Remove nickname** to return to anonymous
- **Not required** - chat works without it

### Auto-Delete Settings
**Choose retention period:**
```
○ 24 hours (default)
○ 7 days
○ 30 days
○ 90 days
```

**Timer shows:**
- Time remaining (e.g., "23h" or "6d")
- Progress bar
- Auto-updates

### Privacy & Data
**What we DON'T collect:**
- ❌ Real names
- ❌ Email addresses
- ❌ Phone numbers
- ❌ Location data (except for clinic finder)
- ❌ Conversation content (stays local)

**What we DO:**
- ✅ Store chat history locally on your device
- ✅ Remember your preferences (nickname, language)
- ✅ Auto-delete conversations after set period
- ✅ Optional anonymous analytics (with your consent)

### Data for Good
**Anonymous Analytics:**
- Helps improve SRHR education
- Cannot be linked to you
- Opt-in/out anytime

**What's tracked (if you opt-in):**
- Popular topic categories (not specific questions)
- Language preferences (aggregated)
- Feature usage (which modules help most)
- Session duration (average time spent)

**Toggle in Settings or Sidebar:**
- ☑️ Share anonymous data (ON)
- ☐ Share anonymous data (OFF)

## 🆘 Emergency Features

### Panic Exit Button
**Location:** Top-right of header (red button)

**When to use:**
- Someone enters the room unexpectedly
- Need instant privacy
- Emergency situations

**What happens:**
1. Tap "Quick Exit" / "Fi hɔ ntɛm" / "Do go kaba"
2. App switches to calculator screen
3. Room 1221 is hidden from view
4. Tap "Return to Room 1221" to come back

**Note:** Chat history is preserved but hidden

### Clear Chat
**Complete conversation deletion:**
1. Open menu → Actions → Clear Chat
2. Confirm the action
3. All messages deleted permanently
4. Cannot be undone

**Use when:**
- You want to start fresh
- Extra privacy needed
- Sharing device with others

## 🔧 Troubleshooting

### Voice Input Not Working
**Problem:** Microphone doesn't activate
**Solutions:**
- Check browser permissions (allow microphone)
- Use Chrome or Edge (best compatibility)
- Ensure internet connection
- Try reloading the page

### Chat Not Sending
**Problem:** Messages don't send
**Solutions:**
- Check internet connection
- Ensure input field has text
- Try reloading the app
- Clear browser cache

### Language Not Changing
**Problem:** Interface stays in wrong language
**Solutions:**
- Tap language button again
- Reload the page
- Clear browser storage
- Check localStorage is enabled

### Timer Not Showing
**Problem:** Session timer missing
**Solutions:**
- Open sidebar to view
- Check auto-delete setting is selected
- Ensure localStorage is enabled
- Start a new chat session
